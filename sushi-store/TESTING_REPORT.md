## 🛠 Техническая реализация

### Настроенные инструменты
- **Jest** - Основной фреймворк тестирования
- **@vue/test-utils** - Утилиты для тестирования Vue компонентов
- **jsdom** - DOM окружение для тестов
- **Babel** - Транспиляция ES6+ кода

### Конфигурация
```json
{
  "testEnvironment": "jsdom",
  "setupFilesAfterEnv": ["<rootDir>/tests/setup.js"],
  "transform": {
    "^.+\\.vue$": "@vue/vue3-jest",
    "^.+\\.js$": "babel-jest"
  }
}
```

## 🚀 Команды для запуска

### Установка зависимостей
```bash
npm install
```

### Запуск всех тестов
```bash
npm test
```

### Запуск конкретных тестов
```bash
# Unit тесты
npm test -- tests/utils.test.js

# Component тесты
npm test -- tests/simple-components.test.js

# Integration тесты
npm test -- tests/simple-integration.test.js
```

### Запуск с покрытием кода
```bash
npm run test:coverage
```

### Запуск в режиме наблюдения
```bash
npm run test:watch
```

### E2E тесты (требует Cypress)
```bash
npx cypress open    # Интерактивный режим
npx cypress run     # Headless режим
```
